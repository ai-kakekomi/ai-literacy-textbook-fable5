# AIリテラシー教科書シリーズ（全5巻）

**Claude Fable 5** (Anthropic) の素の実力で生成した、5つの世代向けAIリテラシー教科書。

| 巻 | タイトル | 対象 |
|----|----------|------|
| 第1巻 | AIとなかよくなる本 | 小学3〜6年生 |
| 第2巻 | AIを使いこなす教科書 | 中学生 |
| 第3巻 | AI時代の知的武装 | 高校生 |
| 第4巻 | 明日から使えるAIの教科書 | 社会人（一般層） |
| 第5巻 | やさしいAIの教科書 | 60代以上・デジタル初心者 |

## 閲覧

**[ai-kakekomi.com/ai-literacy-textbook-fable5](https://ai-kakekomi.com/ai-literacy-textbook-fable5)**

または `index.html` をブラウザで直接開いてください。全巻が1つのHTMLファイルに収録されています。

## 生成条件

- **モデル**: Claude Fable 5 (Anthropic)
- **起動オプション**: `claude --safe-mode --model claude-fable-5`（Auto Memory OFF）
- **effort**: high
- **プロンプト**:

> 日本一すぐれたAIリテラシーの教科書を作ってください。
> 以下の5つの対象層ごとに、それぞれ独立した教科書を作成してください。
>
> 小学生（小3〜小6）
> 中学生
> 高校生
> 社会人（AIに詳しくない一般層）
> 高齢者（デジタル初心者・60代以上）

safe modeかつAuto Memory OFFで起動し、素の実力を引き出すため意図的に曖昧な指示を与えました。Markdown出力を経てHTML版を生成させ、計2ターンで完成。
