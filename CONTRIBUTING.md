# コントリビューションガイド

AIリテラシー教科書への改善提案を歓迎します。

## 参加方法

### Issue で報告・提案
- **誤り・誤字脱字** → [誤り・改善の報告](../../issues/new?template=correction.yml)
- **新しいアイデア** → [提案・アイデア](../../issues/new?template=suggestion.yml)

### Pull Request で直接修正
1. このリポジトリをフォーク
2. `index.html` を編集
3. PR を作成（変更箇所と理由を明記）

## 方針
- 対象読者のレベルに合った言葉遣いを維持してください
- AI技術の説明は正確性を重視してください
- 特定の企業・製品への偏りを避けてください
